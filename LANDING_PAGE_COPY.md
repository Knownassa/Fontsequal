## Features

### Build a library you can use

Browse font families, filter by category or style, preview live specimen text, and keep favorites and project collections close at hand.

### Find the right voice faster

Use focused preview presets, weight controls, spacing and line-height tools, italic variants, and multi-font comparison to make type decisions in context.

### Know what is installed

Scan local fonts, search by family, inspect metadata and paths, identify duplicates, and separate Fontsequal-managed fonts from external ones.

## Local-first

Your font library and settings stay on your machine. Fontsequal stores its data locally and does not require an account or cloud sync. Optional cloud sync is planned for a future release, not this MVP.

## Safe font management

Fontsequal installs and imports fonts only into per-user managed folders. It does not support system-wide installation in v0.1.0, never overwrites originals, and only uninstalls fonts it manages. External and system fonts stay protected.

## Designer preview tools

Turn a font list into a working type lab. Edit preview text, start from practical presets such as brand marks and website heroes, tune size and rhythm, then compare up to four families side by side.

## Roadmap

v0.1.0 establishes safe local workflows for Windows and Linux. Next: stronger Google Fonts catalog controls, richer metadata and exports, deeper collection workflows, and opt-in cloud sync.
